# 🗂️ Profile Manager - Python Application

A Python-based profile management application built to practise real software development workflows.

This project started as a simple command-line application and evolved into a structured application with persistent storage, JSON handling, and a focus on clean development practices.

The goal of this project is not only to build functionality but to understand how software is designed, improved, tested, and maintained.

---

## 🚀 Features

Current features:

✅ Create user profiles
✅ Store profile information
✅ Persistent JSON-based storage
✅ Load existing profiles when application starts
✅ Display saved profiles
✅ Modular function-based structure

# 📌 Git Commit Style

This project follows conventional commit categories:

| Commit   | Purpose          |
| -------- | ---------------- |
| feat     | New feature      |
| fix      | Bug fix          |
| refactor | Code improvement |
| docs     | Documentation    |
| style    | Formatting       |
| test     | Testing          |

Examples:

